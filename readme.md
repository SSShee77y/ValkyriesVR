# ValkyriesVR
A VR aircraft simulation game made with Unity and Oculus SDK. You will need an Oculus / Meta device to play.

### Current Release Version: 0.1.1

- Bigger area to explore
- Scenery pieces (trees, roads, buildings, etc.)
